# Premium Restaurant Website

This is a modern premium restaurant website built for customer demos, portfolio showcase, and business outreach. It includes an immersive restaurant homepage, menu browsing, cart and checkout flow, reservation system, admin dashboard, and WhatsApp ordering functionality.

## Features

- Premium responsive restaurant UI
- Home, Menu, About, Cart, Reservation, and Admin pages
- Dynamic menu and cart system
- Quantity update and order summary
- WhatsApp order placement
- Reservation form
- Admin dashboard for orders, reservations, and menu management
- Firebase setup support
- Mobile-friendly layout
